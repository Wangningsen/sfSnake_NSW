## 运行环境要求

- MinGW-W64 x86_64-msvcrt-mcf-seh 13.0.1 20230402  (experimental)
- git version 2.45.1.windows.1
- cmake version 3.26.3

理论上，解压文件后直接进行build，run，即可运行。不过需要注意的是，尽管使用了gitee的sfml镜像源，还是建议在编译时使用代理（科学上网），否则有可能会导致编译无法通过。
